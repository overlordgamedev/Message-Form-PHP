# Message-Form-PHP

Тестовое задание для компании ООО "Вайт Рэбит".  
Данный проект представляет собой гостевую книгу с отзывами.  
Функционал гостевой книги включает:
1. Ввод данных гостя
2. Базу данных для хранения отзывов
3. Систему CAPTCHA
